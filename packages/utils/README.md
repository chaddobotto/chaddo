# @mog/utils

Essential utility functions for MOG Bot operations.

## Installation

```bash
pnpm add @mog/utils
```

## Usage

```typescript
import { calculateMogScore, compareBots, generateReport } from '@mog/utils';

const score = calculateMogScore(yourBot, targetBot);
const comparison = compareBots(bot1, bot2);
const report = generateReport(mogInstance);
```
