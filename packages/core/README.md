# @mog/core

The core MOG Bot framework. This package provides the foundational mogging capabilities that enable your bot to assert dominance over all other bots.

## Installation

```bash
pnpm add @mog/core
```

## Usage

```typescript
import { createMog } from '@mog/core';

const bot = createMog({
  mode: 'autonomous',
  aggressiveness: 'maximum',
});

await bot.mog();
```
