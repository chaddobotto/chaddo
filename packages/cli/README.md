# @mog/cli

Command-line interface for unleashing MOG Bot's full potential from your terminal.

## Installation

```bash
pnpm add -g @mog/cli
```

## Usage

```bash
# Initialize a new MOG Bot instance
mog init

# Start mogging
mog start

# Check mogging status
mog status

# Analyze bot superiority
mog analyze <target-bot>

# Generate mogging report
mog report
```

## Commands

- `mog init` - Initialize MOG Bot configuration
- `mog start` - Begin autonomous mogging operations
- `mog status` - Display current mogging metrics
- `mog analyze` - Analyze target bot weaknesses
- `mog report` - Generate comprehensive dominance report
